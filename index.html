VirtualBox. La guía definitiva.

VirtualBox es un software de virtualización que permite crear máquinas virtuales en tu equipo. Una máquina virtual es un entorno de trabajo emulado que funciona como un equipo independiente dentro de tu sistema operativo principal.
Con VirtualBox, puedes instalar y ejecutar múltiples sistemas operativos dentro de tu equipo fisico, lo que te permite probar diferentes configuraciones de software, realizar pruebas, crear entornos de desarollo aislados, etc.
Es una herramienta muy util para desarrolladores, probadores de software administradores de sistemas y estudiantes (como yo).
